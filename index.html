Ի՞նչ է տվյալագիտությունը

Տվյալագիտությունը (Data Science) նշանակում է օգտագործել տվյալներ հասկանալու աշխարհը և ճիշտ որոշումներ ընդունելու համար։

Պարզ ասած՝ մենք վերցնում ենք թվեր → վերլուծում ենք → հասկանում ենք ինչ է կատարվում

Կյանքի օրինակներ Ո՞ր ուսանողներն են ավելի լավ սովորում · Ո՞ր խաղն է ամենահայտնի · Ո՞ր ապրանքն է ամենաշատը վաճառվում · Քանի ժամ սովորելիս է գնահատականը բարձրանում
Հարց Ամենալավ համակարգչային խաղը որոշելու համար ի՞նչ տվյալներ կհավաքես: Օրինակ՝ խաղացողների քանակը, թե՞ խաղում անցկացրած ընդհանուր ժամանակը։

Միջին արժեք (Mean)

Սա թվաբանական միջինն է։ Գումարում ենք բոլոր թվերը, բաժանում ենք քանակին։

Բանաձև & Օրինակ Mean = (x₁ + x₂ + … + xₙ) / n

(2 + 4 + 6) / 3 = 4
Փորձիր ինքդ Գտիր mean-ը՝ 3, 7, 10

Միջին անդամ (Median)

Median-ը «կենտրոնի թիվն» է։ Քայլ 1՝ դասավորել թվերը, Քայլ 2՝ վերցնել կենտրոնականը։

Կենտ քանակ 1, 3, 5 → median = 3

Զույգ քանակ 1, 3, 5, 7 → (3 + 5) / 2 = 4
Հարց Median-ը կախվա՞ծ է ամենամեծ թվից:

Ամենահաճախ հանդիպող (Mode)

Mode-ը այն թիվն է, որը ամենաշատն է կրկնվում։

Օրինակ 1, 2, 2, 3 → mode = 2
Փորձիր ինքդ Գտիր mode-ը՝ 4, 4, 5, 6, 6, 6

Outlier (շատ մեծ կամ փոքր թիվ)

Երբ տվյալներից մեկը շատ է տարբերվում մյուսներից, կոչվում է outlier։

Օրինակ` ինչո՞ւ է կարևոր Տրված է՝ 1, 2, 3, 100

Mean = (1+2+3+100)/4 = 26.5 — «խաբվեց» մեծ թվից
Median = (2+3)/2 = 2.5 — մնաց իրականին մոտ

→ Mean-ը կարող է «խաբվել» · Median-ը ավելի կայուն է

Կոռելյացիա (Correlation)

Correlation-ը ցույց է տալիս՝ երկու բան կապված են, թե ոչ։

+1 = ուժեղ դրական կապ 0 = կապ չկա -1 = հակառակ կապ
-10+1
Կարևոր միտք Correlation ≠ պատճառ
Սովորելու ժամեր ↑ → Գնահատական ↑ (~+0.9 դրական կապ)
Խաղալու ժամեր ↑ → Գնահատական ↓ (բացասական կապ)

Նույնիսկ եթե երկու բան կապված են, դա դեռ չի նշանակում,որ մեկը պատճառ է մյուսի։ Օրինակ խաղալը պատճառ չէ ցածր գնահատականի, սակայն սովորաբար շատ խաղացող աշակերտները ավելի քիչ ժամանակ են տրամադրում սովորելուն։

Ստուգիր քեզ

Պատկերացրու դու ուսուցիչ ես Քո աշակերտների գնահատականներն են՝ 5/100, 6/100, 7/100, 8/100, 100/100

Ի՞նչ է Pandas-ը

Pandas-ը Python ծրագրավորման լեզվի գրադարան է։ Այն օգտագործվում է տվյալներ կարդալու (ֆայլերից, օրինակ՝ Excel), տվյալներ մաքրելու, վերլուծելու և հաշվարկներ կատարելու համար։

Կարևոր միտք Pandas-ը առանձին լեզու չէ, այն աշխատում է Python-ի ներսում։

Պարզ ասած՝ Pandas-ը գործիք է տվյալների հետ աշխատելու համար
Ինչ պետք է իմանաս մինչև Pandas Քանի որ Pandas-ը Python-ում է, պետք է իմանաս հիմնական Python-ը՝

փոփոխականներ — x = 5
ցանկեր (lists) — numbers = [1, 2, 3]
ֆունկցիաներ — print(numbers)

Ի՞նչ է Machine Learning-ը

Machine Learning նշանակում է՝ համակարգիչը սովորում է օրինակներից, ոչ թե միայն կանոններից։

Կարևոր տարբերություն Սովորական ծրագրավորում — մենք գրում ենք կանոնները
  եթե միավոր ≥ 50 → անցավ · եթե < 50 → չանցավ

Machine Learning — համակարգիչը գտնում է կանոնները
  տեսնում է շատ ուսանողների տվյալներ → ինքն է սովորում
Հարց Ինչը կընտրեիր՝ ինքդ գրել 100 կանոն, թե՞ թույլ տալ համակարգչին սովորել տվյալներից:

Գծային ռեգրեսիա (Linear Regression)

Սա ամենապարզ ML մոդելներից է։ Գաղափարը՝ գտնել ուղիղ գիծ, որը նկարագրում է տվյալները։

Բանաձև՝ y = a + b·x x → սովորելու ժամեր · y → գնահատական
b → որքան է աճում գնահատականը մեկ ժամ ավելացնելիս
a → «մեկնարկային» արժեք (0 ժամ → a)

Եթե b=10, a=40 → 3 ժամ=70 6 ժամ=100

Model limitation 8 ժամ → 120 — սա իրականում սխալ է, մոդելն «անսահման» է(գնահատականը չի կարող 100-ից բարձր լինել)
Հարց Եթե սովորես 0 ժամ, ի՞նչ կլինի արդյունքը (ըստ մոդելի):
Python · Linear Regression

Որոշման ծառ (Decision Tree)

Այս մոդելը մտածում է հաջորդական հարցերով — նման է մարդու մտածելակերպին:

Ծառի օրինակ Սովորե՞լ ես ≥ 3 ժամ
↳ Այո → լավ արդյունք
↳ Ոչ → Ունե՞ս հիմք
       ↳ Այո → Միջին · Ոչ → Ցածր

Բանկի օրինակ — Ունե՞ս աշխատանք → Եկամուտը բարձր է → վարկն ընդունված է

K-Nearest Neighbors (KNN)

Նոր տվյալը դասվում է իր ամենամոտ «հարևան»-ների հիման վրա:

«Ասա ով է քո ընկերը, ասեմ ով ես դու»

5 ընկերից 4-ը սիրում են ֆիլմը → դու հավանաբար կսիրես → KNN ասում է "Recommend"(խորհուրդ տալ)

Ինչ է անում KNN — գտնում է մոտ տվյալները · նայում նրանց պատասխաններին · ընտրում մեծամասնությունը

Խաղ․ Դու Akinator-ն ես

Պատկերացրու դու այն խաղն ես, որը կարողանում է կռահել, թե ինչ են մտածում

Դու մտածում ես որևէ կերպարի մասին, իսկ խաղը քեզ տալիս է հարցեր, իսկ վերջում… գուշակում է ճիշտը:

Խաղն ունի շատ տվյալներ — հազարավոր մարդիկ պատասխանել են հարցերին, օրինակ՝

Decision Tree (հիմնական գաղափար)
Խաղը աշխատում է հարցերի շղթայով, ամեն հարց → ճյուղ, ամեն պատասխան → նոր ուղղություն:
Իրական մարդ է՞ → Այո → Քաղաքական գործիչ է՞ → Ոչ → Գիտնական է՞

KNN (մոտիկ գաղափար)
Խաղը նաև «հիշում է» նման պատասխանները․ նման պատասխաններ = նման արդյունք»
Եթե դու պատասխանում ես նման մարդկանց, ովքեր մտածել են նույն կերպարի մասին → խաղը կկռահի նույնը

Ինչո՞ւ է այն այդքան ճշգրիտ
Քանի որ ունի շատ տվյալներ՝ յուրաքանչյուր նոր խաղ ավելի խելացի է դառնում(սովորում է տվյալներից)
Փորձիր ինքդ
Մտածիր կերպար՝ սուպերհերոս, YouTuber, կամ խաղի հերոս։ Ի՞նչ 5 հարց կտայիր, որ արագ հասնես պատասխանին:

⚠️ Հետաքրքիր միտք — Եթե սխալ պատասխաններ տաս, խաղը կարող է սխալվել։
Սա ցույց է տալիս՝ Machine Learning-ը կախված է տվյալների որակից

💡 Ամփոփենք միտքը՝ Akinator-ը կախարդանք չէ, դա Decision Tree + KNN + շատ տվյալների ճիշտ օգտագործում է։
Համեմատություն Linear Regression → թիվ է կանխատեսում (գնահ., գին)

KNN → հարևաններով որոշում է (մոտ նմաններ)

Տվյալագիտությունը մեր շուրջն է

Դու արդեն օգտագործում ես տվյալագիտություն ամեն օր՝ YouTube-ը առաջարկում է տեսանյութեր, Google-ը ընտրում է որոնման արդյունքներ, խաղերը հարմարեցվում են քո մակարդակին:

Հարց Դու ես ընտրել այսօրվա դիտելու տեսանյութը, թե՞ համակարգիչն է քեզ առաջարկել:
Բժշկություն Հիվանդությունների վաղ հայտնաբերում · MRI և CT պատկերների վերլուծություն · Քաղցկեղի հայտնաբերում

Երբեմն համակարգիչը նկատում է այն, ինչ մարդը կարող է բաց թողնել, բայց վերջնական որոշումը միշտ բժշկինն է
Ֆինանսներ Խարդախ գործարքների հայտնաբերում · Շուկայի կանխատեսում · Վարկային ռիսկի գնահատում

Եթե քարտով գնում է կատարվում ուրիշ երկրից, ապա համակարգը կարող է դա համարել կասկածելի
ճարտարագիտություն & Ռոբոտներ Ռոբոտների կառավարում · Սենսորների տվյալների վերլուծություն · Գործարաններում որակի ստուգում

Մեքենան կարող է կանխատեսել, որ ինչ-որ մաս շուտով կկոտրվի և նախապես զգուշացնել
Սպորտ Խաղացողների արագություն · Փոխանցումների ճշգրտություն · Գոլերի հավանականություն

Կարող են կանխատեսել ո՞ր թիմն ավելի ուժեղ է, բայց սպորտը անհնար է ճշգրիտ կանխատեսել։ Տվյալները պարզապես օգնում են հասկանալ հավանականությունը
